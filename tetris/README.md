# シンプルテトリス

ブラウザで直接遊べる、純粋な HTML/CSS/JavaScript だけで構成したテトリスです。ビルドや依存インストールは不要で、`index.html` を開けばすぐに遊べます。

## 必要な環境
- モダンブラウザ（Chrome / Edge / Firefox / Safari など）
- 追加インストールは不要です

## 遊び方
1. `index.html` をブラウザで開くとゲームが開始されます。
2. 操作キー
   - ← / → : 横移動
   - ↑ : 右回転
   - ↓ : ソフトドロップ（1 マス素早く落下）
   - スペース : ハードドロップ（着地まで一気に落下）
   - R : リスタート
3. 横 1 列が埋まると消えてスコア + ライン数が加算されます。
4. 新しいミノが出現した直後に既存ブロックと重なってしまうとゲームオーバーです。画面の案内に従って `R` キーで再開できます。

## ファイル構成
- `index.html` : キャンバスと情報パネルを含むページ本体
- `style.css` : レイアウトと簡単なテーマカラー
- `main.js` : フィールド管理、落下ロジック、入力処理、描画処理

## 開発メモ / 拡張のヒント
- テトリミノの形状や色は `main.js` の `TETROMINOES` / `COLORS` で定義しています。形状を変更・追加する場合はここを編集してください。
- `DROP_INTERVAL` を調整すると落下速度を変更できます。レベル制などを導入する場合はこの値を徐々に短くするだけでも簡単な難易度調整になります。
- 衝突判定や行消去ロジックは関数に切り出しているので、ホールド機能・ゴースト表示・スコア計算の変更などを追加しやすい構成です。
